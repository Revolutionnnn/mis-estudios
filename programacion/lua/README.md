## Objetivos de aprendizaje
- Entender la filosofía de diseño de Lua: minimalismo, portabilidad y extensibilidad
- Dominar la sintaxis básica y el modelo de datos (tablas, funciones, closures)
- Comprender metatables, metamétodos y programación orientada a prototipos
- Conocer los ámbitos de uso: videojuegos, sistemas embebidos, scripting de aplicaciones
- Aprender a integrar Lua con C/C++ (API C de Lua)

## Prerrequisitos
- Conocimientos básicos de programación en cualquier lenguaje (variables, condicionales, bucles, funciones)
- Nociones de línea de comandos para ejecutar scripts y usar el REPL
- (Opcional) Familiaridad con C para la sección de integración

## Roadmap
1. Historia, filosofía y ecosistema de Lua
2. Sintaxis fundamental: tipos, variables, operadores, estructuras de control
3. Tablas: la estructura de datos universal
4. Funciones, closures y el modelo de scoping
5. Metatables y metaprogramación
6. Módulos, paquetes y LuaRocks
7. Manejo de errores y corutinas
8. Lua en videojuegos, herramientas y sistemas embebidos
9. Integración con C: la API de Lua
10. LuaJIT y optimización de rendimiento
